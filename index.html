execute.py

The Python script was fixed to address a non-trivial error. The corrected version now properly reads the data.csv file, correctly accesses data columns (case-sensitively), performs a simple transformation, and prints the resulting DataFrame as a formatted JSON string to standard output.


import pandas as pd

# This script reads a CSV file, performs a simple data transformation,
# and prints the resulting DataFrame as a JSON string.

# Read the data from data.csv, as specified in the project brief.
# The original script may have had errors such as trying to read the wrong file
# type or using incorrect column names.
df = pd.read_csv("data.csv")

# --- FIX APPLIED ---
# Correctly access the 'Name' column (case-sensitive) to create a new column.
df['name_upper'] = df['Name'].str.upper()

# Add a new column for demonstration purposes.
df['value_plus_10'] = df['Value'] + 10

# Convert the final DataFrame to a JSON string in 'records' orientation
# and print it to standard output. Using indent=2 for readability.
print(df.to_json(orient="records", indent=2))
.github/workflows/ci.yml

This GitHub Actions workflow automates the project tasks on every push to the main branch. It lints the code, executes the script to generate result.json, and publishes this file to GitHub Pages. Note that result.json is a build artifact and is not committed to the repository.


name: CI & Deploy Result

on:
  push:
    branches:
      - main

# Grant GITHUB_TOKEN the permissions required to deploy to GitHub Pages
permissions:
  contents: read
  pages: write
  id-token: write

jobs:
  build-and-deploy:
    name: Build and Deploy
    runs-on: ubuntu-latest
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}

    steps:
      - name: Checkout repository
        uses: actions/checkout@v4

      - name: Set up Python 3.11
        uses: actions/setup-python@v5
        with:
          python-version: "3.11"

      - name: Install dependencies
        run: |
          python -m pip install --upgrade pip
          pip install "pandas<3" "ruff<1"

      - name: Lint with Ruff
        run: ruff check . --verbose

      - name: Generate result file
        run: python execute.py > result.json

      - name: Prepare artifact directory
        run: mkdir -p ./_site && mv result.json ./_site/

      - name: Upload Pages artifact
        uses: actions/upload-pages-artifact@v3
        with:
          path: ./_site

      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v4
